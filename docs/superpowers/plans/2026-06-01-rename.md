# Note Rename / Move (Link-Aware) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an atomic, link-aware `RenameNote { from, to }` command that moves a note and rewrites `[[wikilinks]]` pointing at it.

**Architecture:** Hexagonal — a new `VaultStore::rename` port capability (+ `PortError::AlreadyExists`), `LocalFsStore::rename` adapter, a pure-domain `rewrite_link_target`, an `Engine::rename_note` use-case over ports, a contract `Command::RenameNote`, a dispatcher arm, and a CLI `rename`.

**Tech Stack:** Rust 1.85, serde + ts-rs (contract), clap (CLI), nextest.

**Verified shapes:** `VaultStore { read, write, delete, list }`; `PortError::{NotFound(String), Adapter(String)}` (thiserror). `From<PortError> for ServiceError` in `cairn-service` matches `NotFound`/`Adapter` **exhaustively** (must gain the new arm). `LocalFsStore { root: PathBuf }` with `fn full(&self, &NotePath)->PathBuf`. `Engine` has `apply_change(&FsChange, &mut dyn EventSink)`, `store`, `NotePath::stem()`, `cairn_ports::FsChange`. `cairn-domain` has `extract_links` in `link.rs`. Contract `Command::{WriteNote,DeleteNote,Commit}`, `CommandResponse::Done`; round-trip test `command_serializes_with_type_tag`. `cairn-service` `dispatch_command` matches `WriteNote`/`DeleteNote`/`Commit`. CLI clap `Command` enum + arms in `run()`.

In `crates/cairn-domain/src/link.rs`, add:
```rust
/// Rewrite `[[from]]` -> `[[to]]` and `[[from|alias]]` -> `[[to|alias]]` in
/// `content`, matching link targets by exact trimmed text. Non-matching links
/// and all other text are left verbatim. Operates on raw content (no frontmatter
/// parsing).
#[must_use]
pub fn rewrite_link_target(content: &str, from: &str, to: &str) -> String {
    let mut out = String::with_capacity(content.len());
    let bytes = content.as_bytes();
    let mut i = 0;
    while i < content.len() {
        if i + 1 < bytes.len() && bytes[i] == b'[' && bytes[i + 1] == b'[' {
            if let Some(close) = content[i + 2..].find("]]") {
                let span_end = i + 2 + close + 2;
                let inner = &content[i + 2..i + 2 + close];
                let (target, alias) = match inner.split_once('|') {
                    Some((t, a)) => (t, Some(a)),
                    None => (inner, None),
                };
                if target.trim() == from {
                    out.push_str("[[");
                    out.push_str(to);
                    if let Some(a) = alias {
                        out.push('|');
                        out.push_str(a);
                    }
                    out.push_str("]]");
                } else {
                    out.push_str(&content[i..span_end]);
                }
                i = span_end;
                continue;
            }
        }
        let ch = content[i..].chars().next().unwrap();
        out.push(ch);
        i += ch.len_utf8();
    }
    out
}
```
Re-export in `crates/cairn-domain/src/lib.rs` (alongside the existing `extract_links` re-export):

```rust
    fn rename(&mut self, from: &NotePath, to: &NotePath) -> Result<(), PortError> {
        let src = self.full(from);
        let dst = self.full(to);
        if !src.exists() {
            return Err(PortError::NotFound(from.as_str().to_string()));
        }
        if dst.exists() {
            return Err(PortError::AlreadyExists(to.as_str().to_string()));
        }
        if let Some(parent) = dst.parent() {
            std::fs::create_dir_all(parent).map_err(|e| PortError::Adapter(e.to_string()))?;
        }
        std::fs::rename(&src, &dst).map_err(|e| PortError::Adapter(e.to_string()))
    }
```

## Task 3: App — `Engine::rename_note`

**Files:** Modify `crates/cairn-app/src/lib.rs`

- [ ] **Step 1: Add the use-case**

In `crates/cairn-app/src/lib.rs`, ensure `rewrite_link_target` is imported from `cairn_domain` (extend the existing `cairn_domain` import to include it). Then add to the `Engine` impl (after `delete_note`):
```rust
    /// Rename/move a note, link-aware. Moves the file, then rewrites links
    /// pointing at the old stem in every note. Emits `NoteDeleted(from)` +
    /// `NoteChanged(to)` + a `NoteChanged` per rewritten note (via `apply_change`).
    ///
    /// # Errors
    /// Propagates `VaultStore::rename` errors (NotFound / AlreadyExists / Adapter).
    pub fn rename_note(
        &mut self,
        from: &NotePath,
        to: &NotePath,
        sink: &mut dyn EventSink,
    ) -> Result<(), PortError> {
        self.store.rename(from, to)?;
        self.apply_change(&FsChange::Removed(from.clone()), sink)?;
        self.apply_change(&FsChange::Changed(to.clone()), sink)?;

        let old_stem = from.stem();
        let new_stem = to.stem();
        if old_stem != new_stem {
            for path in self.store.list()? {
                let raw = self.store.read(&path)?;
                let rewritten = rewrite_link_target(&raw, old_stem, new_stem);
                if rewritten != raw {
                    self.store.write(&path, &rewritten)?;
                    self.apply_change(&FsChange::Changed(path.clone()), sink)?;
                }
            }
        }
        Ok(())
    }
```
(`rewrite_link_target` is `cairn_domain::rewrite_link_target`; add it to the `use cairn_domain::{...}` line.)
